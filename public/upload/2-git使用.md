### 普通上传文件

~~~
git init 初始化
git add read.me 
git commit -m '第一次'
git remote add origin 你的仓库地址
git push -u origin master
~~~



### 创建分支

~~~
git checkout -b	dev 创建dev分支并且切换到
git add 
git commit -m 
git push -u origin dev 上传分支
~~~





### 解决冲突

~~~
git merge

git add .

git push -u origin master
~~~
